Technologies: Java

Project Description:

A console-based Java application for managing dog ownership, allowing users to add, remove, and list dogs and their owners.

The project demonstrates core object-oriented programming principles such as classes, encapsulation, and handling user input efficiently. One challenge was designing a clear and maintainable class structure to manage the relationships between dogs and owners, which I solved through careful planning and organization.

This project strengthened my understanding of Java fundamentals and reinforced good coding practices.
