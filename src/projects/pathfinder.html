Github: https://github.com/malinanita/pathfinder

Technologies: Java, JavaFX

Project Description:

A desktop application to manage and navigate a network of places, allowing users to add locations, create connections, and find optimal paths using a graph data structure. Implemented object-oriented design principles and a user-friendly interface with JavaFX, including file saving and loading functionality.

I collaborated with two teammates using Live Share in Visual Studio Code, which strengthened my teamwork and version control skills.

This project enhanced my understanding of data structures, JavaFX GUI design, and collaborative software development practices.
