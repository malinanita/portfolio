Project Description

A simple application to manage daily tasks, displaying the current date and allowing users to create and delete to-do items, as well as mark tasks as completed. Built with React and JSX using Vite, with CSS modules for scoped styling and LocalStorage to persist tasks across sessions.

This project was a self-driven exercise to deepen my understanding of React, component-based architecture, and frontend best practices. I focused on creating a clean, intuitive, and easy-to-use interface, which I now actually use personally to keep track of my daily tasks.

It strengthened my skills in React, JSX, state management, LocalStorage, modular CSS, and building functional applications from scratch.
