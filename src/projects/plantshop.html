Project Description:

Elm – A plant-inspired e-commerce in PHP & MySQL
A full-featured e-commerce prototype simulating the complete purchase flow – from browsing and filtering products, to cart management, checkout, and order confirmation – all except payment processing.
Built with a strong focus on security (password hashing, prepared statements, session handling, transaction safety) and clean architecture (strict HTML/PHP separation, reusable templates, component-based design).
The responsive frontend was designed in Figma with a calm, nature-inspired aesthetic, and includes features like asynchronous cart updates, category filtering, and server-rendered order detail modals.
